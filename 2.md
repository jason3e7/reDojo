## 32-bit 通用暫存器 ##
* EAX, Accumulator 
* EBX, Base
* ECX, Counter
* EDX, Data

## 32-bit 指標暫存器 ##
* EBP, Base Pointer
* ESP, Stack Pointer
* ESI, Source Index
* EDI, Destination Index

## stack ##
Stack “grows” towards the lower address
ESP at stack top
EBP at stack bottom
